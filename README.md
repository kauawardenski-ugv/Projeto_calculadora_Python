# Projeto_calculadora_Python
Projeto calculadora Python:  linguagem: phiton  calculadora sem interface, somente console  operações: Adição, subtração, multiplicação e divisão.  Porcentagem (%): Fundamental para calcular descontos ou juros rápidos.  Memória: e servem para guardar um resultado temporário e somá-lo ou subtraí-lo de cálculos futuros.
